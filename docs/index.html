Hotkeys

HotkeyAction
Ctrl + OOpen upload dialog
Ctrl + Shift + EOpen "Extension..." menu
/Open "Files, folder, and symbols..." menu (not in text fields)
` or ~Open "Repositories..." menu (not in text fields)
Ctrl + BToggle sidebar
Ctrl + QQuit app
Ctrl + + / - / 0Zoom in/out/reset
Ctrl + ScrollZoom in/out

About

Copilot Desktop is an unofficial Electron-based desktop client for GitHub Copilot, providing a native-feeling experience with robust hotkeys, zoom, tray, and more.
